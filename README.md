**This prioect is a simple social app which implemented with Django framework and has the following features:**
- User registration
- Logging in to your account
- Ability to view personal page
- Create, edit, and delete posts
- Liking posts
- Follow/Unfollow users
- Reset password
- Ability to search based on words in posts
- Considering key points for page security and making changes to user accounts in the admin panel
